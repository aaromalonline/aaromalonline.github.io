Liberate:ATCS - Adaptive Typing & Control System

2025

Liberate:ATCS is an assistive technology project designed for individuals with limited mobility, such as those diagnosed with ALS, muscular dystrophy, or spinal cord injuries. The system translates subtle muscle movements (twitches) into digital control signals.

Hardware Modules

The system utilizes a modular sensor architecture to capture input:

  • Accelerometer-based Module: Uses an ADXL345 sensor to detect precise muscle tremors or twitches.
  • IR Sensor Module: Uses TCRT5000 IR sensors for proximity-based twitch detection, providing a non-contact alternative.
  • Wireless Control: Driven by an ESP32 for real-time, wireless data transmission to the host interface.

Software & Adaptive Interface

The host application, built using Python and PyQt5, features an adaptive keyboard interface. It uses a row-column scanning technique where a single high-gain "click" (decoded from the muscle tremor) selects the desired character or macro. This allows for communication without any traditional keyboard or mouse input.
